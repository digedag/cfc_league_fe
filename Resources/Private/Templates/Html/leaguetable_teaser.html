Template für einen Teaser der Ligatabelle

Das Template hat die volle Funktionaltät der normalen Tabelle. Es kann also auf alle Daten zugegriffen werden. Dieses Template liegt also nur als kleines Beispiel bei.

Vorhandene Marker:

###ROW### Subpart für eine Zeile der Tabelle
###ROW_TYPE### Marker für den Typ einer Zeile. Dies ist eine laufende Nummer bei 0 beginnend und leaguetable.rowtype endend.
###ROW_POSITION_MARK### Marker für spezielle Markierung einer Tabellenposition aus Wettbewerbsdatensatz
###ROW_POSITION### Marker für Tabellenplatz
###ROW_...### Marker für Werte aus dem Teamdatensatz
###ROW_MATCHCOUNT### Marker für Anzahl der Spiele
###ROWS### ###ROW###

###ROW_POSITION###. ###ROW_SHORT_NAME###
