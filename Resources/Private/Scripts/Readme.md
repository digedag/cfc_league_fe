# About

Some possible frontend JS.

# Setup project

* [Install yarn](https://yarnpkg.com/lang/en/docs/install/#debian-stable)
* run `yarn install` in Script directory to install webpack-encore

# How to build

Build **t3sports.js** for production:

```
$ yarn encore production
```

Build **t3sports.js** for develop:

```
$ yarn encore dev --watch
```

Generated `t3sports.js` can be found in folder **Public/JS/t3sports/**.
